# 📊 Customer Churn Dataset Information

## 📌 Overview
This dataset contains customer behavior data used to analyze churn patterns and identify factors that influence customer attrition.

---

## 📁 Dataset Name
customer_churn_dataset.csv

---

## 📊 Description
The dataset includes customer demographic details, service usage patterns, and churn status (whether the customer left the service or not).

---

## 🧾 Key Columns (Typical Structure)
- CustomerID → Unique identifier for each customer  
- Gender → Male / Female  
- Age → Age of customer  
- Tenure → Number of months with the company  
- MonthlyCharges → Monthly billing amount  
- TotalCharges → Total amount charged  
- ContractType → Type of subscription contract  
- InternetService → Type of internet service  
- Churn → Target variable (Yes/No)

---

## 🎯 Objective
To analyze customer behavior and identify patterns that lead to churn using data analytics and visualization techniques.

---

## 📊 Usage
This dataset is used in:
- Python data analysis (Pandas, NumPy)
- Tableau dashboard creation
- Business insights and churn prediction analysis
